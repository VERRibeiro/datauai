Nesta área de Aplicativos, temos um catálogo criado para divulgar todo tipo de aplicativo com interface web, mobile ou até desktop que utilize um ou mais conjuntos de dados disponibilizados no #datapoa.

Se você criou um novo aplicativo que utiliza os conjuntos de dados do #datapoa entre em contato e solicite a inclusão do seu aplicativo neste catálogo.

Envie um email para ifpb@ifpb.edu.br - Coordenação de Inovação contendo os dados do criador (nome, empresa, telefone de contato, ...), o nome do aplicativo, uma breve descrição, a sua URL para acesso ou site e uma URL para sua imagem ou logotipo.
